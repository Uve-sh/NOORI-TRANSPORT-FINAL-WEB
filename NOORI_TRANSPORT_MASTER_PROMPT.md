# NOORI TRANSPORT CO. -- MASTER PROMPT FOR LOVABLE

## OBJECTIVE

Build a COMPLETE mobile-first premium transport company website for
**NOORI TRANSPORT CO.**

This website is intended for REAL CLIENT PRESENTATION.

DO NOT create a generic SaaS website.

DO NOT create a startup landing page.

The website must feel like a long-established Indian logistics company.

The overall information architecture should closely resemble a
traditional transport company website: Hero → Company Overview →
Services → Why Choose Us → Working Process → Fleet → Industries →
Gallery → Clients → Testimonials → Contact → Footer.

## MOST IMPORTANT RULE

DESIGN FOR MOBILE FIRST.

Every design decision should prioritize phones.

Desktop is secondary.

Navigation: - Bottom-friendly tap targets - Sticky top navbar - Large
touch areas (minimum 48px) - Large typography - Comfortable thumb
spacing - Swipeable galleries - Swipeable client logos -
Scroll-triggered animations instead of hover - Hover effects should
gracefully become tap feedback.

## VISUAL STYLE

Premium. Corporate. Industrial. Minimal. Lots of whitespace.

Color palette: Primary #0D3B66 Accent #F15A24 Background #F8FAFC

Typography: Poppins headings Inter body
